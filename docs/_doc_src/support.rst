Support
=======

Support for users and contributors is available through
`GitHub Discussions <https://github.com/hydro-jules/cm4twc/discussions>`_.
Use this support page to find answers to previously asked questions,
to ask questions of your own, and to suggest ideas to improve `cm4twc`.

To report bugs, please use `GitHub Issues <https://github.com/hydro-jules/cm4twc/issues>`_.
