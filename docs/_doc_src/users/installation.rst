Installation
============

If you wish to install the most recent **stable** version of `unifhy`, it is
available on the Python Package Index (PyPI), simply run:

.. code-block:: bash

   python -m pip install unifhy

If you need the **latest**, potentially unstable, features listed in the
:doc:`change log <../changelog>`, please install from the *dev* branch on the
GitHub repository:

.. code-block:: bash

   python -m pip install git+https://github.com/unifhy-org/unifhy.git@dev
